## ⚠️ 보완 및 개선이 필요한 기능

### 1. 날짜 계산 검증 및 수정 로직
**현재 상태**: 프롬프트에만 의존하여 LLM이 날짜를 계산함
**개선 필요**:
- [ ] LLM이 계산한 날짜가 연속적인지 검증하는 함수
- [ ] 날짜 계산이 잘못되었을 때 자동으로 수정하는 로직
- [ ] 주말 제외 옵션 (선택사항)

**구현 위치**: `utils/date_validator.py` (새로 생성)

### 2. JSON 파싱 강화
**현재 상태**: 기본적인 JSON 파싱만 구현됨
**개선 필요**:
- [ ] LLM이 불완전한 JSON을 출력했을 때 부분 파싱
- [ ] 누락된 필드에 대한 기본값 설정
- [ ] JSON 구조 검증

**구현 위치**: `utils/json_parser.py` 개선

### 3. Tavily 검색 최적화
**현재 상태**: Tavily Tool은 생성되었으나 프롬프트에만 의존
**개선 필요**:
- [ ] 카테고리별 최적화된 검색 쿼리 전략
- [ ] 가격 정보 크롤링을 위한 전용 검색 로직
- [ ] 후기 수집을 위한 전용 검색 로직

**구현 위치**: `tool/category_agents.py` 개선 또는 `utils/search_utils.py` 생성

### 4. 에러 처리 강화
**현재 상태**: 기본적인 에러 처리만 구현됨
**개선 필요**:
- [ ] API 호출 실패 시 재시도 로직
- [ ] 타임아웃 처리
- [ ] 상세한 에러 메시지

**구현 위치**: 각 파일에 추가

### 5. 테스트 코드
**현재 상태**: 테스트 코드 없음
**개선 필요**:
- [ ] 각 카테고리별 Agent 단위 테스트
- [ ] JSON 파싱 테스트
- [ ] Word 파일 생성 테스트
- [ ] 통합 테스트

**구현 위치**: `tests/` 디렉터리 생성
